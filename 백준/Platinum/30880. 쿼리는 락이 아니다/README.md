### 문제 설명

<blockquote>
<p style="text-align: center;"><strong>나락도 락이고, 부모님께 온 연락도 락이고, 오락가락?도 락?이지만?, 아쉽게도 쿼리는 락이 아니다.</strong></p>

<p>알파벳 대문자로 이루어진 길이 $N$의 문자열 $S=s_{1} s_{2} \ldots s_{N}$가 주어진다. 이때, 다음과 같은 쿼리를 $Q$번 처리해야 한다.</p>

<ul>
	<li>$1$ $idx$ $c$: $s_{idx}$를 $c$로 변경한다. $(1 \leq idx \leq N,\ c$는 알파벳 대문자$)$</li>
	<li>$2$ $l$ $r$: $s_{l} s_{l+1} \ldots s_{r}$의 부분열 중 <span style="color:#e74c3c;"><code>ROCK</code></span>으로 끝나는 문자열의 개수를 출력한다. 단, 수가 매우 클 수 있으니 $10^9+7$로 나눈 나머지를 출력한다. $(1 \leq l \leq r \leq N)$</li>
</ul>

### 입력 

 <p>첫 번째 줄에 문자열 $S$의 길이 $N$이 주어진다. $(4 \leq N \leq 250000)$</p>

<p>두 번째 줄에 알파벳 대문자로만 이루어진 문자열 $S$가 주어진다.</p>

<p>세 번째 줄에 쿼리의 개수 $Q$가 주어진다. $(1 \leq Q \leq 250000)$</p>

<p>네 번째 줄부터 $Q$개의 줄에 걸쳐 쿼리가 주어진다. 가장 마지막으로 주어지는 쿼리는 $2$번 쿼리이다.</p>

### 출력 

 <p>$2$번 쿼리에 대해 정답을 한 줄에 하나씩 출력한다.</p>
